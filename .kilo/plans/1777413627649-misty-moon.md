# План: Массовая синхронизация VPN Sources из Plain Text

## Краткое резюме

Добавить административный endpoint `PUT /api/v1/admin/vpn-sources/sync-text` для массовой синхронизации VPN URI из plain text через Swagger UI. Решение должно быть простым как замена строк в txt-файле, но безопасным для уже выданных подписок.

## Критический вывод о подписках

**Подписки хранят только `vpn_source_id`, не snapshot URI.** Это означает:
- ✅ Деактивация (`is_active=false`) безопасна — источник исключится из всех подписок
- ✅ Новые URI автоматически появятся в подписках с соответствующими тегами
- ❌ Физический DELETE нельзя использовать — разрушит связи `subscription_issue_items`
- ✅ Режим `replace` с `deactivate_missing=true` — оптимальный вариант

**Логика:**

1. **Парсинг txt:**
   - Разбить на строки
   - Пропустить пустые и комментарии (#)
   - Для каждой строки: извлечь URI, fragment (#name), номер строки
   - Валидировать URI через существующий `CompositeVpnUriValidator`

2. **Режим replace:**
   - Получить все активные источники из `import_group`
   - Новые URI → создать
   - существующие URI → обновить (name, tags, is_active=true)
   - отсутствующие URI → deactivate (is_active=false) если `deactivate_missing=true`

3. **Режим upsert:**
   - Существующие URI → обновить
   - Новые URI → создать
   - Старые не трогать

4. **Режим append:**
   - Только новые URI → создать
   - Существующие пропустить

5. **Dry run:**
   - Все расчёты без записи в БД
   - Вернуть preview

6. **Транзакционность:**
   - Если `ignore_invalid=false` и есть ошибки → откатить всё
   - Если `dry_run=false` → применять в одной транзакции

7. **Запись истории (NEW):**
   - Создать `VpnSourceImport` entity с результатами
   - Сохранить через `import_repo.create()`
   - Даже dry_run записывается в историю (с пометкой dry_run=true)

### 4.3 URI Parser Service (в infrastructure)

**Файл:** `src/infrastructure/parsers/plain_text_uri_parser.py`

```python
class PlainTextUriParser:
    MAX_LINES = 500
    
    def parse(self, text: str) -> list[ParsedUriDTO]:
        lines = text.strip().split("\n")
        if len(lines) > self.MAX_LINES:
            raise ValueError(f"Too many lines: max {self.MAX_LINES}")
        
        result = []
        for i, line in enumerate(lines, start=1):
            stripped = line.strip()
            if not stripped or stripped.startswith("#"):
                continue
            
            # Extract fragment (#name)
            uri_part, fragment = self._extract_fragment(stripped)
            
            result.append(ParsedUriDTO(
                line_number=i,
                raw_uri=uri_part,
                name=fragment,
                is_valid=True,
                error=None,
            ))
        
        return result
    
    def _extract_fragment(self, uri: str) -> tuple[str, str | None]:
        # URL fragment after # is the name
        if "#" in uri:
            parts = uri.split("#", 1)
            return parts[0], parts[1] if parts[1] else None
        return uri, None
    
    def _mask_uri_for_logging(self, uri: str) -> str:
        # Hide sensitive parts: password, uuid
        scheme = uri.split("://")[0] if "://" in uri else ""
        return f"{scheme}://***MASKED***"
```

from fastapi import Body, Query

@router.put(
    "/vpn-sources/sync-text",
    response_model=SyncTextResponse,
    summary="Sync VPN sources from plain text",
    description="...",
)
async def sync_vpn_sources_text(
    text: str = Body(
        ...,
        media_type="text/plain",
        examples={
            "example": {
                "summary": "Example VPN list",
                "value": (
                    "# Main servers\n"
                    "vless://uuid-1@example.com:443?security=reality#Amsterdam-1\n"
                    "vless://uuid-2@example.com:443?security=reality#Amsterdam-2\n"
                    "\n"
                    "# Backup\n"
                    "trojan://password@example.com:443?security=tls#Warsaw-1"
                ),
            },
        },
    ),
    admin: str = Depends(get_current_admin),
    vpn_source_repo: VpnSourceRepository = Depends(get_vpn_source_repo),
    tag_repo: VpnSourceTagRepository = Depends(get_tag_repo),
    validator: VpnUriValidator = Depends(get_validator),
    tags: Annotated[str | None, Query(description="Comma-separated tags")] = None,
    import_group: Annotated[str, Query()] = "default",
    mode: Annotated[str, Query()] = "replace",
    dry_run: Annotated[bool, Query()] = True,
    deactivate_missing: Annotated[bool, Query()] = True,
    ignore_invalid: Annotated[bool, Query()] = False,
    name_strategy: Annotated[str, Query()] = "fragment",
):
    tag_list = tags.split(",") if tags else []
    
    use_case = SyncVpnSourcesTextUseCase(
        vpn_source_repo=vpn_source_repo,
        tag_repo=tag_repo,
        validator=validator,
    )
    
    result = await use_case.execute(
        text=text,
        mode=mode,
        import_group=import_group,
        tags=tag_list,
        dry_run=dry_run,
        deactivate_missing=deactivate_missing,
        ignore_invalid=ignore_invalid,
        name_strategy=name_strategy,
    )
    
    return SyncTextResponse(
        dry_run=result.dry_run,
        mode=result.mode,
        import_group=result.import_group,
        tags=result.tags,
        parsed_count=result.parsed_count,
        valid_count=result.valid_count,
        invalid_count=result.invalid_count,
        to_create_count=result.to_create_count,
        to_update_count=result.to_update_count,
        to_deactivate_count=result.to_deactivate_count,
        created=[SyncTextPreviewItem(...) for item in result.created],
        updated=[SyncTextPreviewItem(...) for item in result.updated],
        deactivated=[SyncTextPreviewItem(...) for item in result.deactivated],
        failed=[SyncTextFailureResponse(...) for item in result.failed],
    )

## Этап 6: Безопасность

### 6.1 Redaction в логах

- Использовать `_mask_uri_for_logging()` для всех URI в логах
- Hide: UUID in vless userinfo, password in trojan, method:password in ss

### 6.2 Ограничения

- Max lines: 500
- Max body size: FastAPI уже ограничивает по конфигу
- dry_run=true по умолчанию
- При invalid > 0 и ignore_invalid=false → HTTP 400 с preview

### 6.3 HTTP Errors

| Код | Ситуация |
|-----|----------|
| 400 | Невалидные строки при ignore_invalid=false |
| 413 | Тело превышает лимит |
| 422 | Невалидные query параметры |
| 401 | Без авторизации |
